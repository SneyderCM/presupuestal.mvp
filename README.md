
## ¿Qué hace?
- Registra ingresos y gastos sin internet
- Muestra saldo actual y resumen mensual
- Guarda datos localmente con `localStorage`

## Cómo usarlo
1. Abre `index.html` en tu navegador (Chrome, Edge, etc.)
2. Registra transacciones
3. Los datos se guardan automáticamente

## Tecnologías
- HTML5
- CSS3
- JavaScript puro
- Persistencia: `localStorage`

## Notas
- No requiere servidor ni instalación
- Funciona 100% offline
